# **App Name**: TidyTag

## Core Features:

- Container Creation: Allow users to create named storage containers with custom names.
- NFC Tag Linking: Link storage containers to NFC tags using the device's NFC reader.
- Item View: Enable users to view item name, image, quantity and date last added within a scanned container.
- Item Addition: Allow users to add new items to a container via a simple input form. The tool can suggest potential items that might go into that container, based on container name and items already present.
- Item Removal: Remove items from a container using a selection interface.
- Local Data Storage: Persistent local storage for all container and item data. Data persists between sessions and app restarts.
- Container Management: Add and remove storage containers.

## Style Guidelines:

- Primary color: Navy blue (#000080) to represent organization and reliability.
- Background color: Light blue (#E0F2F7) to provide a clean and simple feel.
- Accent color: Sky blue (#87CEEB) for buttons and interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) for a clean, readable interface.
- Use simple, flat icons to represent storage containers and items.
- Prioritize a clean and intuitive layout for easy navigation and quick access to features.
- Subtle animations, like fade-in effects and smooth transitions, to enhance user experience without being distracting.
